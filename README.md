🚀 Wood Ninja

A fun and interactive wood-cutting game inspired by arcade-style mechanics


---

📌 Overview

Wood Ninja is an engaging 2D game where players slice wooden logs with precision and timing, similar to fruit-cutting arcade games. The project focuses on smooth gameplay mechanics, animations, and user interaction, making it a great demonstration of game development concepts.


---

🎮 Features

✂️ Real-time slicing mechanics

🪵 Dynamic wooden log objects

🎯 Score tracking system

⚡ Smooth animations and transitions

🎨 Clean and minimal UI design

🔄 Interactive gameplay experience

2️⃣ Install Dependencies

(or npm install if JS)

3️⃣ Run the Game

python main.py

(or npm start / open index.html)

🧠 How It Works

Logs appear dynamically on the screen

Player interacts using mouse/touch/keyboard

When sliced, logs split into parts

Score updates based on accuracy and timing
